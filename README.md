# minnami-try-act
try&amp;act muvies

## 動画サイトの使い方

### メインページ (`index.html`)
- 動画の **再生 / 一時停止** ボタンで動画を操作できます
- **⏪ 10秒戻る** / **10秒進む ⏩** ボタンで 10 秒単位のシーク操作ができます
- プログレスバーをクリックして任意の位置にジャンプできます
- ページ下部の **感想フォーム** に名前と感想を入力して送信してください

### 管理者ページ (`admin.html`)
- 右上の「管理者ページ」リンクからアクセスします
- パスワードを入力してログインすると、視聴者から送られた感想一覧を確認できます
- **初期パスワード**: `minnami2024`（必要に応じて `script.js` 内の `ADMIN_PASSWORD` を変更してください）

> **注意**: このサイトは静的ファイルのみで動作するため、感想データはブラウザの `localStorage` に保存されます。
> 本番運用の場合はバックエンドサーバーの導入を推奨します。

### 動画ファイルの配置
`index.html` と同じディレクトリに `video.mp4` を配置すると動画が再生されます。
